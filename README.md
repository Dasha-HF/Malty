# Maltynation

Одностраничный адаптивный сайт по предоставленному PDF-макету.
Страница собрана из отдельных редактируемых секций и оригинальных изображений.
Зеленые и розовые SVG-ленты используются как разделители между слайдами.

Добавлены спокойные интерактивные эффекты:

- короткая фирменная заставка;
- плавное появление контента при прокрутке;
- повторное появление контента при прокрутке вверх и вниз;
- мягкое нативное притяжение к ближайшему смысловому экрану;
- дополнительная доводка до секции при остановке рядом с ее началом;
- прорисовка лент-разделителей;
- легкий параллакс иллюстраций и банок;
- анимированные показатели;
- индикатор прогресса страницы.
- компактный навигатор по разделам, раскрывающийся при наведении или нажатии.
- крупная рекламная типографика, фоновые слова и спокойная бренд-строка.

Контакты и кнопка покупки сейчас заполнены временными данными.

Добавлен смысловой раздел «Зачем здесь солод?» с объяснением роли
светлого и темного солода во вкусе лимонада.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```sh
python3 -m http.server 8000
```

После этого сайт будет доступен по адресу `http://localhost:8000`.
